Tentang Robot

- Robot masuk ke website dan melakukan login dengan menggunakan username dan password yang sudah disiapkan.

- Pada halaman product robot memproses submit data yang diimput dari file excel.

- Membuat log yang berisi hasil proses dari setiap baris data yang disubmit.

- Pada halaman asset, robot memproses download dokumen, dimana parameter tanggal yang disubmit sudah ditentukan.

- File yang didownload tersimpan di local PC dengan nama file “Asset_yyyyMMdd.xls”, dimana yyyyMMdd merupakan nilai konversi tanggal hari ini.

- Robot menghitung total nilai dari semua asset, serta total quantity dan total value untuk beberapa AssetType data dari excel asset sebelumnya
